# media drop zone

Drop photos / videos into the matching folder — first file found is used, rest go into bonus strip.

## folders

| folder | frame | what to drop |
|---|---|---|
| `01-day-one/` | Jun 15 2025 · day zero | first pic / rain day |
| `02-early-nights/` | Jul 2025 · nights | long-call screenshots, anything soft |
| `03-dakshinchitra/` | Aug 2025 · dakshinchitra | the tape day pics |
| `04-chikmagalur/` | Oct 2025 · chikmagalur | stairs, coffee, hoodie |
| `05-year-end/` | Dec 2025 · year end | NYE / december together |
| `06-valentine/` | Feb 2026 · valentine | that boring-but-spectacular date |
| `07-us-now/` | Mar 2026 · us, now | recent pics |
| `08-today/` | Apr 24 2026 · today | fresh bday shots (can be empty) |
| `bonus/` | extra reel frames | anything left over, in name-sorted order |

## formats

- images: `.jpg` `.jpeg` `.png` `.webp` `.avif` `.gif`
- videos: `.mp4` `.mov` `.webm`

filenames sort alphabetically — name them `01.jpg`, `02.mp4` etc. if order matters.
